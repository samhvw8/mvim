# install

git clone https://github.com/samhvw8/mvim.git ~/.config/nvim

## Install on MacOS

```bash
brew install neovim

pip install  pynvim

brew install ripgrep
brew install fzf bat

pip install jedi
```

Note: fzf preview need check bat theme

## file .bashrc or .zshrc

```bash
alias vim="nvim"
```

## In Vim

Run command `:PlugInstall`
